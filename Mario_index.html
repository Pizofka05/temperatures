2. Data Cleaning: Clean the data by handling missing values, outliers, and any inconsistencies in the dataset that may affect the analysis.

Command

cleaned_data <- weather %>%

  filter(!is.na(wind_gust))

boxplot(weather$wind_gust, main = "Boxplot of wind_gust")

Q1 <- quantile(weather$wind_gust, 0.25, TRUE)

Q3 <- quantile(weather$wind_gust, 0.75, TRUE)

IQR <- Q3 - Q1

lower_bound <- Q1 - 1.5 * IQR

upper_bound <- Q3 + 1.5 * IQR

cleaned_data <- cleaned_data %>%

  filter(wind_gust >= lower_bound & wind_gust <= upper_bound)

negative_values <- cleaned_data$wind_gust < 0

cleaned_data$wind_gust[negative_values] <- NA

 

Output

 

Explain the command

1.     Cleaned_data is a command to remove data with missing “wind_gust” values

2.     Boxplot is to visualize distribution of “wind_gust”

3.     Q1, Q3 and IQR is to calculate the interquartile range

4.     Lower_bound and upper_bound is to define upper and lower bounds for outlier

5.     Negative_values is to check the negative values

 

Explain the output

That is the graph of boxplot of missing value in wind_gust
